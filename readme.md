A very simple appium / android / node test.  All you need is here.

### getting started

You must have 
1. docker
2. docker-compose
3. node

## APK

Your apk should be in ```sdks``` folder. 
in docker-compose.yml change sdk name

### launch appium and android emulator
Launch ```docker-compose up```

that's all.

### launch test:

```npm i && npm start```
